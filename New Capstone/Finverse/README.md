# Finserve 

Finserve is a simple frontend web project based on a digital finance and international money transfer platform idea.  
This project is created using only **HTML, CSS and basic JavaScript**.

Users can:
- Create an account 
- Login 
- Link or unlink a bank account (demo toggle)
- Enter money transfer details
- See currency conversion and tax calculation
- View recent transfer logs
